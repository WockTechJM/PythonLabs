# PythonLabs

This repository contains my Python practice scripts and notes from my first Python lab on Mac.  

**Contents:**

- `test.py` → My first Python script demonstrating `print()` and `input()`.
- `command_log.txt` → Logs outputs from the script.
- `notes.md` → Notes and observations about what I learned.
- `scripts/` → Folder to organize Python scripts (optional).
- `screenshots/` → Folder to store any screenshots from my practice.
- `venv/` → Python virtual environment (for safe package management).

**Objective:**  
Learn Python basics, virtual environment usage, and logging script outputs for practice and portfolio purposes.
